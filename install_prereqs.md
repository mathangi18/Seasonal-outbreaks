# Project Prerequisites

## Required Software

1.  **AnyLogic Personal Learning Edition (PLE)**
    *   **Version:** 8.9.6
    *   **Download:** [AnyLogic Website](https://www.anylogic.com/downloads/)

2.  **Java Development Kit (JDK)**
    *   **Version:** 1.8 (Java 8)
    *   **Note:** Required by AnyLogic 8.9.x. OpenJDK or Oracle JDK 8 are both acceptable.
    *   **Verification:** Run `javac -version` in your terminal. It should output something like `javac 1.8.0_xxx`.

3.  **Git**
    *   **Purpose:** Version control and repository management.
    *   **Download:** [Git Website](https://git-scm.com/downloads)

4.  **Python 3 (Optional)**
    *   **Purpose:** Running the automated validation scripts (`repo_validation/`).
    *   **Download:** [Python Website](https://www.python.org/downloads/)
