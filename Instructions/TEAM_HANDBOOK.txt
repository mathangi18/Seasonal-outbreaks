How to open AnyLogic model:
1. Download seasonaloutbreaks_al.zip from this folder.
2. Unzip it anywhere.
3. Open AnyLogic PLE 8.9.x.
4. File → Open Model → Select the .alp file inside the unzipped folder.
5. The model should run immediately if Java 8 (OpenJDK 1.8) is installed.

Git workflow:

- Submit PR to main.
